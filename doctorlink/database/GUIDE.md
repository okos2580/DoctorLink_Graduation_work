# SQL Server Management Studio에서 스크립트 실행하기

## 자세한 단계별 가이드

### 1단계: SQL Server Management Studio 실행

1. Windows 시작 메뉴에서 "SQL Server Management Studio"를 검색하여 실행합니다.

### 2단계: SQL Server에 연결

1. 서버 유형: "데이터베이스 엔진" 선택
2. 서버 이름: 로컬 서버면 "localhost" 또는 컴퓨터 이름\인스턴스명 입력
3. 인증: "SQL Server 인증" 선택
4. 사용자 이름: "sa" 입력
5. 비밀번호: SQL Server 설치 시 설정한 비밀번호 입력
6. "연결" 버튼 클릭

### 3단계: 데이터베이스 스키마 스크립트 실행

1. 메뉴에서 "파일(File)" > "열기(Open)" > "파일(File)" 선택
2. "create_database.sql" 파일을 찾아서 선택하고 "열기(Open)" 클릭
3. 스크립트 내용이 쿼리 편집기에 표시됩니다.
4. "실행(Execute)" 버튼을 클릭하거나 F5 키를 눌러 스크립트 실행
5. 하단의 메시지 창에 "데이터베이스 스키마가 성공적으로 생성되었습니다." 메시지가 표시되는지 확인

### 4단계: 저장 프로시저 스크립트 실행

1. 새 쿼리 창을 열려면 "파일(File)" > "새로 만들기(New)" > "쿼리 (빈 쿼리 편집기)(Query with Current Connection)" 선택
2. 메뉴에서 "파일(File)" > "열기(Open)" > "파일(File)" 선택
3. "create_stored_procedures.sql" 파일을 찾아서 선택하고 "열기(Open)" 클릭
4. 스크립트 내용이 쿼리 편집기에 표시됩니다.
5. "실행(Execute)" 버튼을 클릭하거나 F5 키를 눌러 스크립트 실행
6. 하단의 메시지 창에 "저장 프로시저가 성공적으로 생성되었습니다." 메시지가 표시되는지 확인

### 5단계: 결과 확인

1. 개체 탐색기(Object Explorer)에서 "데이터베이스(Databases)" 폴더를 펼칩니다.
2. "DoctorLink" 데이터베이스를 찾아 펼칩니다.
3. "테이블(Tables)" 폴더를 펼쳐 모든 테이블이 생성되었는지 확인합니다.
   - 다음과 같은 테이블들이 있어야 합니다: Users, Patients, Doctors, Hospitals, Appointments 등
4. "프로그래밍 기능(Programmability)" > "저장 프로시저(Stored Procedures)" 폴더를 펼쳐 모든 저장 프로시저가 생성되었는지 확인합니다.
   - 다음과 같은 저장 프로시저들이 있어야 합니다: usp_CreateUser, usp_CreatePatient, usp_CreateAppointment 등

## 문제 해결

### 스크립트 실행 중 오류가 발생하는 경우

1. **오류 메시지 확인**: 메시지 창에 표시된 오류 메시지를 자세히 읽어보세요.
2. **데이터베이스 존재 여부 확인**: DoctorLink 데이터베이스가 이미 존재하는지 확인하세요. 이미 존재한다면, 해당 부분은 건너뛰게 됩니다.
3. **권한 문제**: SQL Server에 충분한 권한이 있는 계정으로 로그인했는지 확인하세요. 가급적 'sa' 계정이나 관리자 권한이 있는 계정을 사용하세요.
4. **데이터베이스가 사용 중인 경우**: 다른 연결에서 데이터베이스를 사용 중이면 일부 작업이 실패할 수 있습니다. 다른 연결을 닫고 다시 시도하세요.

### 테이블이나 저장 프로시저가 보이지 않는 경우

1. 개체 탐색기에서 DoctorLink 데이터베이스를 마우스 오른쪽 버튼으로 클릭하고 "새로 고침(Refresh)"을 선택하세요.
2. 여전히 보이지 않는다면, 스크립트 실행 시 오류가 발생했을 수 있습니다. 메시지 창의 오류 로그를 확인하세요.

## 추가 팁

- 테이블이나 저장 프로시저를 확인하려면 각 항목을 마우스 오른쪽 버튼으로 클릭하고 "스크립트 생성(Script as)" > "CREATE to" > "새 쿼리 편집기 창(New Query Editor Window)"을 선택하면 생성 스크립트를 볼 수 있습니다.
- 특정 테이블의 데이터를 확인하려면 테이블을 마우스 오른쪽 버튼으로 클릭하고 "상위 1000개 행 선택(Select Top 1000 Rows)"을 선택하세요.

# 졸업논문 마크다운 파일 사용 방법

## 생성된 파일 목록

1. `졸업논문_섹션1_서론.md` - I. 서론 (개발 배경, 목적, 개발도구 등)
2. `졸업논문_섹션2_요구분석.md` - II. 요구분석 (용어정의, 기능 분석)
3. 추가 섹션들은 대화를 계속하면 생성됩니다

## 한글(HWP) 파일로 변환하는 방법

### 방법 1: 직접 복사-붙여넣기 (가장 간단)

1. 각 `.md` 파일을 메모장이나 VSCode로 엽니다
2. 내용 전체를 복사 (Ctrl+A → Ctrl+C)
3. 한글(HWP)을 실행하고 `졸업논문.hwp` 파일을 엽니다
4. 원하는 위치에 붙여넣기 (Ctrl+V)
5. 제목은 **굵게**, 코드는 `고정폭 글꼴`로 서식 조정

### 방법 2: Pandoc 사용 (Word 경유)

```bash
# Pandoc 설치 (https://pandoc.org/installing.html)

# 마크다운 → Word 변환
pandoc 졸업논문_섹션1_서론.md -o 섹션1.docx

# Word → HWP 변환: Word에서 열어서 "다른 이름으로 저장" → HWP 형식 선택
```

### 방법 3: Typora 사용 (추천)

1. Typora 다운로드 및 설치 (https://typora.io/)
2. `.md` 파일을 Typora로 엽니다
3. 상단 메뉴: **파일** → **내보내기** → **PDF** 또는 **Word**
4. Word 파일을 한글에서 열어 HWP로 저장

### 방법 4: VSCode + Markdown PDF 확장

1. VSCode에 "Markdown PDF" 확장 설치
2. `.md` 파일을 VSCode로 엽니다
3. 우클릭 → **Markdown PDF: Export (pdf)** 선택
4. PDF를 한글에 삽입하거나, Word로 변환 후 HWP로 저장

## 서식 가이드

### 마크다운 → 한글 서식 대응표

| 마크다운 | 한글 서식 |
|----------|-----------|
| `# 제목` | 제목 1 (24pt, 굵게) |
| `## 제목` | 제목 2 (20pt, 굵게) |
| `### 제목` | 제목 3 (16pt, 굵게) |
| `**굵게**` | 굵게 |
| `*기울임*` | 기울임 |
| `` `코드` `` | 고정폭 글꼴 (Consolas, 맑은 고딕 Mono) |
| ` ```코드블록``` ` | 코드 블록 (배경색 회색, 고정폭 글꼴) |
| `- 목록` | 글머리 기호 |
| `1. 목록` | 번호 매기기 |
| `| 표 |` | 표 삽입 |

### 코드 블록 서식 (한글에서)

```
1. 코드 블록 선택
2. 서식 → 글꼴 → "Consolas" 또는 "D2Coding"
3. 서식 → 문단 → 배경색: 연한 회색 (#F5F5F5)
4. 왼쪽 여백: 1cm
```

### 표 서식 (한글에서)

```
1. 표 삽입: 표 → 표 만들기
2. 첫 행 배경색: 파란색 (#4472C4)
3. 첫 행 글자색: 흰색
4. 테두리: 모든 테두리 표시
```

## 분량 늘리기 팁

현재 약 15장 → 30~50장으로 확장하려면:

1. **스크린샷 추가**
   - 웹/모바일 앱 실행 화면
   - 데이터베이스 ERD
   - API 테스트 결과 (Postman)
   - 시스템 아키텍처 다이어그램

2. **코드 예시 추가**
   - 주요 소스코드 전체 삽입 (이미 제공됨)
   - 코드 라인당 주석 설명

3. **표와 차트 추가**
   - 기능 비교표
   - 테스트 케이스 표
   - 예약 통계 그래프

4. **부록 확장**
   - 전체 API 명세서
   - 데이터베이스 테이블 정의서
   - 테스트 케이스 목록

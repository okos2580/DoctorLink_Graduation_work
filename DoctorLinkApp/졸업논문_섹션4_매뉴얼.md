## 2. 사용방법

### 2.1 웹 애플리케이션 사용 가이드

#### 2.1.1 회원가입 및 로그인

**회원가입**

1. 웹사이트 접속: `http://localhost:3000` (또는 배포 URL)
2. 상단 메뉴에서 "회원가입" 클릭
3. 회원가입 정보 입력:
   - 이메일 주소 (로그인 ID로 사용)
   - 비밀번호 (8자 이상, 영문/숫자/특수문자 포함)
   - 비밀번호 확인
   - 이름
   - 전화번호 (010-1234-5678 형식)
   - 생년월일 (선택)
   - 성별 (선택)
4. "회원가입" 버튼 클릭
5. 가입 완료 메시지 확인 후 자동 로그인

**로그인**

1. 웹사이트 접속
2. 상단 메뉴에서 "로그인" 클릭
3. 이메일과 비밀번호 입력
4. "로그인" 버튼 클릭
5. 메인 페이지로 이동

**로그아웃**

1. 우측 상단 프로필 아이콘 클릭
2. "로그아웃" 메뉴 선택

#### 2.1.2 병원 검색

**키워드 검색**

1. 상단 메뉴에서 "병원 찾기" 클릭
2. 검색창에 병원명, 주소, 또는 진료과 입력
   - 예: "서울대학교병원", "강남구", "내과"
3. Enter 키 또는 검색 버튼 클릭
4. 검색 결과 목록 확인

**필터 사용**

1. 병원 찾기 페이지에서 "필터" 버튼 클릭
2. 원하는 조건 선택:
   - **병원 유형**: 종합병원, 병원, 의원, 치과, 한의원
   - **진료과**: 내과, 외과, 정형외과, 소아과, 산부인과 등
   - **정렬**: 거리순, 평점순, 이름순, 리뷰 많은 순
3. "적용" 버튼 클릭
4. 필터링된 결과 확인

**병원 상세 정보 보기**

1. 검색 결과에서 병원 카드 클릭
2. 병원 상세 정보 확인:
   - 기본 정보 (주소, 전화번호, 진료 시간)
   - 소속 의사 목록
   - 환자 리뷰 및 평점
   - 카카오맵 위치
3. "전화하기" 버튼으로 즉시 통화
4. "예약하기" 버튼으로 예약 페이지 이동

#### 2.1.3 예약하기

**예약 프로세스**

1. 병원 상세 페이지에서 "예약하기" 버튼 클릭
2. **Step 1: 의사 선택**
   - 원하는 의사의 "선택" 버튼 클릭
   - 의사의 전문 분야 및 경력 확인
3. **Step 2: 날짜 선택**
   - 캘린더에서 원하는 날짜 클릭
   - 토요일/일요일은 병원에 따라 선택 불가
4. **Step 3: 시간 선택**
   - 예약 가능한 시간 슬롯 표시 (녹색)
   - 이미 예약된 시간은 회색으로 비활성화
   - 원하는 시간 클릭
5. **Step 4: 증상 입력**
   - 방문 사유 입력 (예: "감기 증상", "건강검진")
   - 추가 메모 입력 (선택)
6. "예약 확정" 버튼 클릭
7. 예약 완료 메시지 및 예약 번호 확인

**예약 시 주의사항**
- 예약은 1시간 단위로 가능합니다
- 당일 예약은 3시간 이후부터 가능합니다
- 예약 변경/취소는 예약 시간 1시간 전까지 가능합니다

#### 2.1.4 예약 관리

**예약 내역 조회**

1. 상단 메뉴에서 "예약 관리" 클릭
2. 나의 예약 목록 확인:
   - 예약 날짜 및 시간
   - 병원명 및 의사명
   - 예약 상태 (대기, 승인, 완료, 취소)

**예약 필터링**

1. 예약 관리 페이지에서 상태 필터 선택:
   - **대기 중**: 승인 대기 중인 예약
   - **승인됨**: 병원에서 승인한 예약
   - **완료**: 진료 완료된 예약
   - **취소됨**: 취소된 예약
2. 검색창에 병원명 또는 의사명 입력하여 검색

**예약 상세 보기**

1. 예약 카드 클릭
2. 예약 상세 정보 확인:
   - 예약 번호
   - 병원 정보 (주소, 전화번호, 지도)
   - 의사 정보
   - 예약 날짜 및 시간
   - 방문 사유

**예약 수정**

1. 예약 상세 페이지에서 "수정" 버튼 클릭
   - 조건: 예약 시간 2시간 전까지만 수정 가능
2. 날짜 또는 시간 변경
3. "수정 완료" 버튼 클릭
4. 수정 확인 메시지

**예약 취소**

1. 예약 상세 페이지에서 "취소" 버튼 클릭
   - 조건: 예약 시간 1시간 전까지만 취소 가능
2. 취소 사유 선택 (선택적)
3. "취소 확정" 버튼 클릭
4. 취소 확인 메시지

#### 2.1.5 진료 기록 조회

**진료 기록 목록**

1. 상단 메뉴에서 "진료 기록" 클릭
2. 나의 진료 기록 목록 확인:
   - 진료 날짜
   - 병원명 및 의사명
   - 진단명
   - 처방전 유무

**진료 기록 상세**

1. 진료 기록 카드 클릭
2. 상세 정보 확인:
   - 진단명 (Diagnosis)
   - 치료 내용 (Treatment)
   - 처방전 (Prescription)
   - 의사 메모 (Notes)
   - 첨부 파일 (검사 결과, X-ray 등)

**진료 기록 검색**

1. 검색창에 진단명, 치료 내용, 또는 메모 키워드 입력
2. Enter 키 클릭
3. 검색 결과 확인

**진료 기록 인쇄**

1. 진료 기록 상세 페이지에서 "인쇄" 버튼 클릭
2. 브라우저 인쇄 대화상자에서 "인쇄" 클릭

#### 2.1.6 마이페이지

**프로필 정보 수정**

1. 우측 상단 프로필 아이콘 클릭
2. "프로필 편집" 메뉴 선택
3. 수정 가능한 정보:
   - 이름
   - 전화번호
   - 주소
   - 생년월일
   - 성별
4. "저장" 버튼 클릭

**의료 정보 관리**

1. 마이페이지에서 "의료 정보" 탭 클릭
2. 수정 가능한 정보:
   - 혈액형
   - 알레르기 (복수 입력 가능)
   - 복용 중인 약
   - 과거 병력
   - 비상 연락처
3. "저장" 버튼 클릭

**비밀번호 변경**

1. 마이페이지에서 "보안" 메뉴 클릭
2. 현재 비밀번호 입력
3. 새 비밀번호 입력 (8자 이상)
4. 새 비밀번호 확인
5. "변경" 버튼 클릭

**알림 설정**

1. 마이페이지에서 "알림 설정" 메뉴 클릭
2. 알림 유형별 ON/OFF:
   - 예약 알림
   - 리마인더 알림
   - 시스템 공지
   - 프로모션 알림
3. 변경 사항 자동 저장

#### 2.1.7 리뷰 작성

**리뷰 작성 조건**
- 진료 완료된 예약에 대해서만 작성 가능
- 예약 1건당 1개의 리뷰만 작성 가능

**리뷰 작성 방법**

1. 진료 기록 목록에서 "리뷰 작성" 버튼 클릭
2. 평점 선택 (1~5점, 별점)
3. 리뷰 내용 입력 (최소 10자 이상)
4. "등록" 버튼 클릭
5. 등록 완료 메시지 확인

**리뷰 수정/삭제**

1. 병원 상세 페이지에서 나의 리뷰 확인
2. "수정" 또는 "삭제" 버튼 클릭
3. 수정 시: 평점 및 내용 변경 후 "저장"
4. 삭제 시: "삭제 확정" 클릭

#### 2.1.8 고객 지원

**공지사항 확인**

1. 하단 메뉴에서 "공지사항" 클릭
2. 공지사항 목록 확인
3. 제목 클릭하여 상세 내용 확인

**FAQ (자주 묻는 질문)**

1. 하단 메뉴에서 "FAQ" 클릭
2. 카테고리 선택:
   - 회원가입/로그인
   - 예약
   - 결제
   - 기타
3. 질문 클릭하여 답변 확인

**1:1 문의하기**

1. 하단 메뉴에서 "문의하기" 클릭
2. 문의 정보 입력:
   - 문의 유형 (일반, 예약, 기술, 제안)
   - 이름
   - 이메일
   - 전화번호 (선택)
   - 제목
   - 문의 내용
3. "문의 등록" 버튼 클릭
4. 등록 완료 메시지 및 자동 회신 이메일 확인

**문의 내역 확인**

1. 마이페이지에서 "문의 내역" 메뉴 클릭
2. 나의 문의 목록 확인
3. 문의 카드 클릭하여 상세 내용 및 답변 확인

### 2.2 모바일 애플리케이션 사용 가이드

#### 2.2.1 앱 설치

**Android**

1. Google Play 스토어 접속
2. 검색창에 "DoctorLink" 입력
3. 검색 결과에서 "DoctorLink - 의료 예약" 선택
4. "설치" 버튼 터치
5. 설치 완료 후 "열기" 터치

**iOS**

1. App Store 접속
2. 검색 탭에서 "DoctorLink" 입력
3. 검색 결과에서 "DoctorLink - 의료 예약" 선택
4. "받기" 버튼 터치
5. Face ID / Touch ID / Apple ID 비밀번호 인증
6. 설치 완료 후 "열기" 터치

**초기 설정**

1. 앱 최초 실행 시 권한 요청:
   - 알림 권한 (권장)
   - 위치 권한 (선택 - GPS 기반 검색 시 필요)
   - 카메라 권한 (선택 - 프로필 사진 촬영 시 필요)
2. 권한 허용 또는 거부 선택
3. 로그인 화면 표시

#### 2.2.2 회원가입 및 로그인

**회원가입**

1. 로그인 화면에서 "회원가입" 버튼 터치
2. 회원정보 입력 (웹과 동일)
3. "회원가입" 버튼 터치
4. 가입 완료 후 자동 로그인

**로그인**

1. 이메일과 비밀번호 입력
2. "로그인" 버튼 터치
3. (선택) "자동 로그인" 체크박스 선택
4. 홈 화면으로 이동

**생체 인증 로그인 (지문/Face ID)**

1. 마이페이지 > 설정 > 보안
2. "생체 인증 사용" 토글 ON
3. 기기의 생체 인증 설정 (설정 앱에서)
4. 다음 로그인부터 지문 또는 얼굴 인식으로 빠른 로그인

#### 2.2.3 홈 화면

**홈 화면 구성**

1. **상단 섹션**
   - 사용자 인사말
   - 알림 아이콘 (새 알림 시 빨간 뱃지)
   
2. **빠른 액션 버튼**
   - 병원 찾기
   - 예약하기
   - 진료 기록
   - 응급실 찾기

3. **건강 팁**
   - 오늘의 건강 정보
   - 좌우 스와이프로 더 보기

4. **근처 병원**
   - GPS 기반 주변 병원 5개 표시
   - 터치하여 상세 정보 확인

5. **최근 예약**
   - 다가오는 예약 2개 표시
   - "모두 보기"로 예약 관리 페이지 이동

**하단 탭 바**
- 홈
- 검색 (병원 찾기)
- 예약
- 진료 기록
- 마이 (마이페이지)

#### 2.2.4 병원 검색 (GPS 기반)

**자동 위치 기반 검색**

1. 하단 탭에서 "검색" 터치
2. 위치 권한 허용 시 자동으로 현재 위치 주변 병원 표시
3. 지도에 병원 마커 표시 (선택적)

**수동 검색**

1. 상단 검색창 터치
2. 병원명, 주소, 또는 진료과 입력
3. 검색 결과 목록 스크롤

**필터 및 정렬**

1. 검색 결과 화면에서 "필터" 버튼 터치
2. 하단 시트에서 조건 선택:
   - 병원 유형
   - 진료과
   - 거리 (1km, 3km, 5km, 10km)
3. "적용" 버튼 터치
4. 정렬 버튼 터치하여 정렬 기준 변경

**병원 상세 정보**

1. 병원 카드 터치
2. 병원 상세 화면 표시:
   - 기본 정보
   - 소속 의사 목록 (가로 스크롤)
   - 리뷰 및 평점
   - 카카오맵 미리보기
3. 하단 액션 버튼:
   - 전화하기 (즉시 통화 앱 실행)
   - 길찾기 (카카오맵 또는 지도 앱 실행)
   - 예약하기

#### 2.2.5 예약하기

**예약 프로세스 (모바일)**

1. 병원 상세 화면에서 "예약" 버튼 터치
2. **Step 1: 의사 선택**
   - 의사 목록에서 선택
   - 의사 카드 터치하여 상세 프로필 확인
3. **Step 2: 날짜 선택**
   - 캘린더에서 날짜 터치
   - 좌우 스와이프로 월 변경
4. **Step 3: 시간 선택**
   - 시간 슬롯 그리드 표시
   - 녹색: 가능, 회색: 불가
   - 원하는 시간 터치
5. **Step 4: 증상 입력**
   - 방문 사유 텍스트 입력
   - (선택) 추가 메모 입력
6. "예약 완료" 버튼 터치
7. 성공 다이얼로그 표시

**예약 알림 설정**

- 예약 생성 시 자동으로 알림 설정
- 예약 1일 전, 1시간 전 푸시 알림 발송

#### 2.2.6 예약 관리

**예약 목록**

1. 하단 탭에서 "예약" 터치
2. 예약 목록 확인 (최신순)
3. 상태별 필터 탭:
   - 전체
   - 대기 중
   - 승인됨
   - 완료
   - 취소됨

**예약 상세**

1. 예약 카드 터치
2. 상세 정보 화면:
   - 예약 번호 (QR 코드)
   - 병원 정보
   - 의사 정보
   - 날짜 및 시간
   - 방문 사유
3. 하단 액션 버튼:
   - 병원에 전화하기
   - 길찾기
   - 예약 수정
   - 예약 취소

**예약 수정**

1. 예약 상세 화면에서 "수정" 버튼 터치
2. 날짜 또는 시간 재선택
3. "수정 완료" 버튼 터치
4. 토스트 메시지로 수정 확인

**예약 취소**

1. 예약 상세 화면에서 "취소" 버튼 터치
2. 확인 다이얼로그에서 "취소 확정" 터치
3. (선택) 취소 사유 선택
4. 취소 완료 토스트 메시지

#### 2.2.7 진료 기록

**진료 기록 목록**

1. 하단 탭에서 "진료 기록" 터치
2. 상단 요약 카드:
   - 총 진료 수
   - 처방전 수
   - 첨부 파일 수
3. 진료 기록 목록 (최신순)
4. 아래로 스크롤하여 더 보기

**진료 기록 상세**

1. 진료 기록 카드 터치
2. 상세 정보 화면:
   - 진료 날짜
   - 병원 및 의사 정보
   - 진단명
   - 치료 내용
   - 처방전 (접기/펼치기)
   - 의사 메모
   - 첨부 파일 (이미지 갤러리)
3. 첨부 파일 터치하여 전체 화면으로 보기
4. 공유 버튼으로 진료 기록 공유 (PDF)

**진료 기록 검색**

1. 상단 검색창 터치
2. 진단명, 치료 내용, 또는 메모 입력
3. 실시간 검색 결과 표시

#### 2.2.8 알림

**알림 목록**

1. 홈 화면 상단 벨 아이콘 터치
2. 알림 목록 확인:
   - 예약 관련 (파란색 아이콘)
   - 리마인더 (주황색 아이콘)
   - 시스템 공지 (회색 아이콘)
   - 프로모션 (녹색 아이콘)
3. 아래로 스크롤하여 더 보기

**알림 읽기**

1. 알림 카드 터치
2. 관련 화면으로 이동:
   - 예약 알림 → 예약 상세
   - 공지사항 → 공지사항 상세
3. 자동으로 읽음 표시

**알림 삭제**

1. 알림 카드를 왼쪽으로 스와이프
2. "삭제" 버튼 터치
3. 또는 상단 "모두 읽음" / "모두 삭제" 버튼 터치

#### 2.2.9 마이페이지

**프로필 편집**

1. 하단 탭에서 "마이" 터치
2. 프로필 헤더에서 "편집" 버튼 터치
3. 프로필 사진 터치하여 변경:
   - 카메라로 촬영
   - 갤러리에서 선택
4. 개인정보 및 의료정보 수정
5. 상단 "저장" 버튼 터치

**설정**

1. 마이페이지에서 "설정" 메뉴 터치
2. 설정 항목:
   - 알림 설정
   - 보안 (비밀번호 변경, 생체 인증)
   - 언어 선택 (한국어/English)
   - 앱 버전 정보
   - 로그아웃

**고객 지원**

1. 마이페이지에서 "고객 지원" 섹션
2. 메뉴 선택:
   - 공지사항
   - FAQ
   - 1:1 문의
   - 고객센터 전화
   - DoctorLink 소개

**1:1 문의**

1. "1:1 문의" 터치
2. 문의 정보 입력
3. "문의 등록" 버튼 터치
4. 문의 내역은 "문의 내역" 메뉴에서 확인

### 2.3 관리자 기능 사용 가이드

#### 2.3.1 관리자 대시보드

**접근 방법**

1. 관리자 계정으로 로그인
2. 자동으로 관리자 대시보드로 이동

**대시보드 구성**

1. **주요 통계 카드**
   - 총 사용자 수
   - 총 병원 수
   - 총 예약 수
   - 오늘의 예약 수

2. **차트 및 그래프**
   - 주간 예약 추이 (막대 그래프)
   - 사용자 유형 분포 (파이 차트)
   - 예약 상태 현황 (도넛 차트)

3. **빠른 액션**
   - 사용자 관리
   - 병원 관리
   - 예약 관리
   - 알림 발송

#### 2.3.2 사용자 관리

**사용자 목록 조회**

1. 대시보드에서 "사용자 관리" 클릭
2. 사용자 목록 테이블 확인:
   - ID, 이름, 이메일, 역할, 상태, 가입일

**사용자 검색 및 필터**

1. 검색창에 이름 또는 이메일 입력
2. 역할 필터 선택:
   - 전체
   - 환자
   - 의사
   - 관리자

**사용자 상세 정보**

1. 사용자 행 클릭 또는 "상세" 버튼 클릭
2. 상세 정보 확인:
   - 기본 정보
   - 예약 내역
   - 진료 기록 (환자인 경우)
   - 로그인 이력

**사용자 정보 수정**

1. 사용자 상세 페이지에서 "수정" 버튼 클릭
2. 수정 가능한 항목:
   - 이름
   - 전화번호
   - 역할 (patient/doctor/admin)
   - 상태 (active/inactive)
3. "저장" 버튼 클릭

**사용자 계정 비활성화**

1. 사용자 상세 페이지에서 "비활성화" 버튼 클릭
2. 확인 다이얼로그에서 "확인" 클릭
3. 비활성화된 사용자는 로그인 불가

#### 2.3.3 병원 관리

**병원 목록**

1. 대시보드에서 "병원 관리" 클릭
2. 병원 목록 테이블 확인

**병원 등록**

1. "병원 추가" 버튼 클릭
2. 병원 정보 입력:
   - 병원명, 주소, 전화번호
   - 병원 유형, 진료과
   - 위도/경도 (지도에서 선택)
   - 진료 시간
3. "등록" 버튼 클릭

**병원 정보 수정**

1. 병원 목록에서 "수정" 버튼 클릭
2. 정보 수정
3. "저장" 버튼 클릭

**의사 관리**

1. 병원 상세 페이지에서 "의사 관리" 탭 클릭
2. "의사 추가" 버튼으로 새 의사 등록
3. 의사 정보 입력:
   - 연결할 사용자 계정 선택
   - 전문 분야
   - 면허 번호
   - 경력, 학력, 소개
4. "저장" 버튼 클릭

#### 2.3.4 예약 관리

**전체 예약 조회**

1. 사이드바에서 "예약 관리" 클릭
2. 전체 예약 목록 확인
3. 날짜 범위 선택하여 필터링

**예약 상태 변경**

1. 예약 목록에서 예약 선택
2. 상태 드롭다운에서 변경할 상태 선택:
   - 승인 (pending → approved)
   - 완료 (approved → completed)
   - 취소 (→ cancelled)
3. 자동으로 환자에게 알림 발송

#### 2.3.5 알림 발송

**전체 사용자 알림**

1. 대시보드에서 "알림 발송" 클릭
2. 알림 정보 입력:
   - 제목
   - 내용
   - 알림 유형 (system/promotion)
3. "전체 발송" 버튼 클릭
4. 확인 다이얼로그에서 "확인" 클릭

**개별 사용자 알림**

1. 사용자 상세 페이지에서 "알림 보내기" 버튼 클릭
2. 알림 정보 입력
3. "발송" 버튼 클릭

**알림 발송 이력**

1. 사이드바에서 "알림 이력" 클릭
2. 발송 내역 테이블 확인:
   - 발송 일시, 제목, 수신자 수, 읽음 수

#### 2.3.6 콘텐츠 관리

**공지사항 관리**

   - 제목

3. FAQ 작성:
   - 카테고리 선택
   - 질문
   - 답변
4. "등록" 버튼 클릭

**문의 관리**

1. 사이드바에서 "문의 관리" 클릭
2. 답변 대기 중인 문의 목록 확인
3. 문의 선택하여 상세 내용 확인
4. 답변 입력 후 "답변 등록" 버튼 클릭
5. 자동으로 사용자 이메일로 답변 발송

## 3. 문제 해결 (Troubleshooting)

### 3.1 웹 애플리케이션 문제

**로그인이 안 돼요**

- 이메일과 비밀번호를 정확히 입력했는지 확인
- Caps Lock이 켜져 있는지 확인
- 브라우저 쿠키 및 JavaScript가 활성화되어 있는지 확인
- 비밀번호를 잊었다면 "비밀번호 찾기" 클릭

**페이지가 로딩되지 않아요**

- 인터넷 연결 상태 확인
- F5 키를 눌러 페이지 새로고침
- 브라우저 캐시 삭제 (Ctrl + Shift + Delete)
- 다른 브라우저에서 시도

**예약이 안 돼요**

- 로그인 상태 확인
- 선택한 날짜와 시간이 예약 가능한지 확인
- 브라우저 콘솔(F12)에서 에러 메시지 확인
- 고객센터로 문의

### 3.2 모바일 애플리케이션 문제

**앱이 실행되지 않아요**

- 기기를 재시작
- 앱을 삭제 후 재설치
- 기기의 OS 버전이 요구사항을 충족하는지 확인
- 저장 공간이 충분한지 확인

**GPS가 작동하지 않아요**

- 설정 > 앱 > DoctorLink > 권한에서 위치 권한 허용
- 기기의 위치 서비스가 켜져 있는지 확인
- 실외에서 시도 (실내는 GPS 신호 약함)

**푸시 알림이 오지 않아요**

- 설정 > 앱 > DoctorLink > 알림 설정 확인
- 앱 내 설정 > 알림 설정에서 각 알림 유형 활성화 확인
- 기기의 방해 금지 모드 해제

**로그아웃이 자주 돼요**

- "자동 로그인" 체크박스 선택
- 앱 데이터를 삭제하지 않도록 주의
- 안정적인 네트워크 연결 유지
